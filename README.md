# 졸업프로젝트 스타트
이화여자대학교 컴퓨터공학과 졸업프로젝트 스타트 200OK 팀의 레포지토리입니다. 

## 프로젝트 개요 
200OK은 고령층 사회복지 AI 상담 플랫폼을 기획하는 팀입니다. 
고령층을 위한 음성 기반 AI 상담 서비스를 통해 고령층이 보다 사회서비스를 잘 이용할 수 있도록 돕습니다.
