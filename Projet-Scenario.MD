| **프로젝트 키워드** | AI 문제 자동 생성, PDF 기반 학습자료 분석, 개인화 문제 추천, 학생 오답 패턴 분석, 게임 기반 학습, 실시간 대시보드, FastAPI·Spring Boot Microservice |

| **무엇을 만들고자 하는가** | 본 프로젝트는 **교사의 입력 최소화 → AI 자동 생성 → 게임 기반 학습 적용 → 개인화 복습맵 제공**의 전체 학습 사이클을 자동화하는 교육용 플랫폼입니다. 교사는 단원명·PDF·학습 목표만 입력하면, AI가 해당 내용에 맞는 문항을 생성하고, 수업 중 학생 정답률과 오답 패턴을 실시간 분석합니다. 수업 종료 후에는 학생별 취약 개념을 자동 진단하여 개인 맞춤형 복습맵과 피드백 리포트를 제공합니다. 또한 생성된 문제들은 Phaser 기반 미니게임에 즉시 반영되어 학생 참여도를 극대화합니다.   |
| **고객** | **Persona 1: 초등학교 교사 A** 나이/직업: 34세 / 초등 교사문제점: 하루 1~2시간씩 직접 문제 제작·편집해야 하며, 학생별 개별 피드백 제공이 어려움 목표: 수업 준비 시간을 단축하고, 학생별 취약 개념을 빠르게 파악하고 싶음기술 친화도: LMS·온라인 학습 플랫폼 사용에 익숙함동기 요소: 자동화·업무 효율화, 데이터 기반 학습 분석. <br> **Persona 2: 학생** 나이/직업: 11세 / 초등학생문제점: 지루한 문제풀이, 낮은 집중력, 반복 학습에 대한 거부감목표: 재미있게 문제를 풀고, 게임 요소 속에서 자연스럽게 학습하고 싶음 기술 친화도: 모바일/PC 게임에 익숙함동기 요소: 캐릭터, 보상(코인·뱃지), 미니게임, 경쟁·퀘스트 구조 |
| **Pain Point** |  - 교사는 매 수업마다 새로운 문제지를 제작하기에 시간이 부족 → **AI 기반 자동 문제 생성 필요**- 학생별 오답 원인 파악이 어려워 개별 피드백 제공이 제한됨 → **오답 패턴 분석 엔진 필요**- 현 교육 소프트웨어의 개인화 기능 부족 → **취약 개념 기반 자동 복습맵 생성 필요**- 학생들은 전통적 문제풀이 방식에 지루함 → **게임형 학습 인터페이스 필요**- 단원별 난이도 조절 문제 존재 → **학습 데이터 기반 난이도 조절 AI 필요**  |
| **사용할 소프트웨어 패키지의 명칭과 핵심기능/용도** | ① **Python(FastAPI)** – AI 모듈 서버 구현, 문제 생성 및 학습 데이터 분석 API 제공<br>② **ChatGPT API, RAG** – GPT 기반 자동 문제 생성 및 학생 맞춤형 학습 피드백 제공<br>③ **React** – 문제 제작 화면 UI 및 교사용 대시보드 UI, 학생용 학습 UI 구현<br>④ **Phaser.js** – 보드게임형 미니게임 시스템 및 게임 화면 구현<br>⑤ **Spring Boot + MySQL** – 사용자, 세션, 퀘스트 데이터 관리 및 백엔드 로직 처리<br>⑥ **Redis + WebSocket** – 학습 세션 실시간 동기화 및 진행 현황 공유 |
